# Professional Finder React Express App

See the finished project running on Google Cloud here: https://professional-finder-app.appspot.com/

To use:
- npm install
- npm run webpack
- npm start

Watchers:
- npm run webpack:watch
- npm run test:watch

To test:
- npm test
